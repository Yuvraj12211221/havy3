Demonstrating HAVY Chatbot Widget Integration

Welcome to Our Sample Site

This is a demonstration page showing how to integrate the HAVY Chatbot Widget into any website. Look for the chat button in the bottom-right corner!

How to Integrate the Widget

Follow these simple steps to add the chatbot to your website:

<!-- Step 1: Configure the chatbot before loading the script -->
<script>
  window.HAVYChatbotConfig = {
    rasaServerUrl: 'http://localhost:5005',  // Your Rasa server URL
    userId: 'user',                          // Unique user identifier
    position: 'bottom-right',                // 'bottom-right' or 'bottom-left'
    primaryColor: '#3B82F6'                  // Your brand color
  };
</script>

<!-- Step 2: Load the chatbot widget script -->
<script src="/chatbot-widget.js"></script>
📝 Note: Make sure your Rasa server is running and accessible at the specified URL. The widget will automatically connect to it when a user sends their first message.

Configuration Options

rasaServerUrl

The URL of your Rasa server endpoint. Default: http://localhost:5005

userId

Unique identifier for the user session. Default: 'user'

position

Widget position: 'bottom-right' or 'bottom-left'

primaryColor

Hex color code for the widget theme. Default: '#3B82F6'

Try It Out!

Click the chat button in the bottom-right corner to start a conversation with the chatbot. Ask questions, get help, or just say hello!
